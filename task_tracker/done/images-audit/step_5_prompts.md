## Структура работы

### 5.1 Hero P0 — 4–6 вариантов

Использовать базовые сценарии из ТЗ, адаптировав под Тянь-Шань:

- **A. Dawn Summit Silhouette** — одинокий силуэт альпиниста на заснеженном гребне, Хан-Тенгри в фоне, первые лучи из-за горы
- **B. Tent at Golden Hour** — лагерь с одной палаткой на альпийском лугу, Хан-Тенгри на заднем плане defocused, людей нет
- **C. Hiker Descending** — фигура спускается по моренному склону с рюкзаком, долина в золотом свете
- **D. Warrior's Horizon** — крупный план лица в профиль (казахские черты, обветренная кожа), bokeh гор
- **E. Ridge Line Minimal** — минималистичный пейзаж: изрезанный гребень, тёплый градиент неба, никого
- **F. Return to Camp** — две фигуры возвращаются к лагерю на закате, длинные тени, масштаб гор

Выбрать **4–6 самых сильных** и для каждого написать полный промпт. Если какой-то сценарий не тянет (дублирует другой или слабый для hero) — заменить своим, но **с обоснованием** в комментарии.

Формат каждого промпта:

```markdown
### hero-main-A — Dawn Summit Silhouette

**Target file:** `public/images/home/hero-a.webp`
**Aspect:** 16:9 ultra-wide cinematic (желательно 21:9 если hero полноэкранный)
**Priority:** P0
**Chain from:** None

**Prompt:**
> Ultra-wide cinematic 16:9 frame. A single mountaineer silhouette...
> [single paragraph 120–200 слов, включает: scene → subject → setting → lighting → palette → composition → style references → film look → negatives]
> [в конец вставить Style anchor block из Step 4]

**Acceptance:**
- силуэт мал в кадре, не заполняет >15% высоты
- Хан-Тенгри узнаваем (пирамидальный снежный пик)
- свет тёплый, не пересвечен
- нет видимых логотипов

**Rejection:**
- фигура слишком крупная / позирует
- палитра ушла в vivid оранж
- пик выглядит как Маттерхорн (швейцарский)
```

## Общие правила для всех промптов

1. **Style-block из Step 4** — обязательно в конце каждого промпта
2. **Ориентация словами** — "ultra-wide cinematic 16:9 frame", "square composition", "tall portrait 3:4 orientation"
3. **Локация-якорь** — Tian Shan / Khan Tengri / Kazakhstani всегда упоминается
4. **Негативные указания** — дублируются даже если уже в style-block (усиливают)
5. **Длина промпта** — 120–250 слов. Длиннее не значит лучше, nano-banana теряет фокус
6. **Acceptance / Rejection** — обязательны для каждой картинки
7. **Target file path** — указывать всегда, конвенция `public/images/<section>/<name>.webp`
